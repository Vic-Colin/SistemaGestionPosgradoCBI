🔘 Botones

El sistema cuenta con diferentes estilos de botones para diversas acciones:

Botón Primario

Para acciones principales (guardar, agregar, exportar)

.btn-primario {
    background: #CD032E;
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 8px;
    font-weight: bold;
    transition: all 0.3s ease;
}

.btn-primario:hover {
    background: #a50225;
    transform: translateY(-2px);
}

Botón Secundario (Outline)

Para acciones secundarias (cancelar, regresar, salir)

.btn-secundario {
    background: white;
    color: #CD032E;
    border: 2px solid #CD032E;
    padding: 10px 20px;
    border-radius: 8px;
    font-weight: bold;
    transition: all 0.3s ease;
}

.btn-secundario:hover {
    background: #CD032E;
    color: white;
}

✅ Mejores Prácticas

1. Consistencia: Usa siempre las mismas clases para elementos similares en toda la aplicación.
2. Responsive: Asegura de que los componentes se vean bien en diferentes tamaños de pantalla.
3. Accesibilidad: Incluye estados hover, focus y asegura buen contraste de colores.
4. Performance: Evita selectores CSS muy complejos que puedan ralentizar el renderizado.
