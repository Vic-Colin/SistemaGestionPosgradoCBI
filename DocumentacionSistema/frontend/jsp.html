Documentación de Vistas JSP

Sistema de Gestión de Posgrado CBI - UAM Azcapotzalco

index.jsp

Propósito: Página de autenticación y login del sistema.

Características:
  • Formulario de autenticación de usuarios
  • Invoca al LoginServlet para validación de credenciales
  • Validación mediante hash SHA-256 de contraseñas
  • Creación de sesión HTTP tras autenticación exitosa
  • Redirección al dashboard según rol de usuario
  • Diseño responsive con logos institucionales UAM y CBI

Elementos: Login Autenticación Seguridad

dashboard.jsp

Propósito: Menú principal del sistema que proporciona acceso a todos los módulos.

Características:
  • Verificación de sesión activa (redirige a index.jsp si no hay sesión)
  • Visualización del nombre de usuario y rol activo
  • Acceso rápido a módulos: Alumnado, Becas, Profesorado y Tesis
  • Botón de cierre de sesión (LogoutServlet)
  • Marca de agua con logo institucional
  • Diseño con navegación superior roja institucional

Elementos: Dashboard Navegación Menú Principal

Alumnos.jsp

Propósito: Gestión completa del módulo de alumnado del posgrado.

Características:
  • Listado dinámico de alumnos con filtrado en tiempo real
  • Consulta de datos desde AlumnadoServlet
  • Formulario modal para registro y edición de alumnos
  • Campos: datos personales, académicos, beca SECIHTI, titulación y proyecto de tesis
  • Filtros por: matrícula, trimestre de ingreso, trimestre pierde calidad, año de titulación, estatus UAM
  • Exportación de reportes a PDF (jsPDF + autoTable)
  • Confirmación de eliminación de registros
  • Carga dinámica de catálogos (áreas de concentración, profesores)
  • Acordeones para secciones opcionales (beca, titulación, tesis)
  • Validación de campos obligatorios

Elementos: CRUD Filtros PDF Modal

Becas.jsp

Propósito: Consulta y seguimiento de becas SECIHTI/CONAHCYT.

Características:
  • Listado de alumnos con información de becas
  • Datos desde BecaServlet con JOIN a tabla alumno
  • Filtros por: CVU, generación (trimestre de ingreso), estatus de beca
  • Visualización de fechas: fin de beca, fecha máxima CONAHCYT, titulación
  • Exportación de reportes a PDF
  • Búsqueda en tiempo real con debounce (300ms)
  • Generación dinámica de opciones de generación (últimos 10 años)

Elementos: Becas CONAHCYT Filtros PDF

Profesorado.jsp

Propósito: Consulta del personal académico del núcleo de posgrado.

Características:
  • Listado de profesores registrados
  • Datos desde ProfesoradoServlet
  • Filtros por: número económico y CVU
  • Visualización de: nivel SNI, tipo de dedicación, departamento, fecha de ingreso, correo institucional
  • Exportación de reportes a PDF
  • Búsqueda en tiempo real con debounce
  • Encabezado de tabla con estilo rojo institucional

Elementos: Profesores SNI Filtros PDF

Proyectos.jsp

Propósito: Consulta de proyectos de tesis e investigación del posgrado.

Características:
  • Listado de proyectos de tesis registrados
  • Datos desde ProyectosServlet con múltiples JOIN
  • Filtros por: matrícula, título de tesis, director/codirector
  • Visualización de: estatus UAM del alumno, área de concentración
  • Exportación de reportes a PDF
  • Búsqueda en tiempo real con debounce
  • Indicadores visuales de estatus (colores por estado)

Elementos: Tesis Proyectos Filtros PDF

Relación entre Componentes

Arquitectura MVC:

  • Modelo: Clases Java en uam.azc.sistemagestionposgradocbi.modelo
  • Vista: Archivos JSP (documentados arriba)
  • Controlador: Servlets en uam.azc.sistemagestionposgradocbi.servlets
  • DAO: Acceso a datos en uam.azc.sistemagestionposgradocbi.DAO

Tecnologías Utilizadas

  • Backend: Java EE 8, Servlets, JSP, JSTL, JDBC, MySQL
  • Frontend: HTML5, CSS3, JavaScript (ES6)
  • Librerías: Gson (JSON), jsPDF + autoTable (PDF), Bootstrap (estilos)
  • Servidor: Apache Tomcat
  • Base de Datos: MySQL (mcc_uam_azcapotzalco)
